# Certificate Example

This example demonstrates how to configure custom TLS/SSL certificates for a Container Apps Managed Environment. It shows both direct certificate upload and Key Vault reference patterns.

> **Note**: This example requires a valid certificate. See Prerequisites below.

## Prerequisites

- A valid TLS/SSL certificate in PFX or PEM format
- Azure Key Vault with the certificate stored (for Key Vault reference pattern)
- Appropriate Key Vault access policies configured
